样式嵌套

最后，有时候你会希望应用一个样式在一大段文字上，但里面可能也包含了子段落的样式。就像 HTML 里的 "<b>Hello <i>world</i></b>"，整段是粗体但包含了一部分斜体的。

现在我们的 API 还不支持这样，所以让我们来加上它。思路是允许将一串 Style… 不止应用在 String 上，还能应用在已经存在属性的 AttrString 上。
